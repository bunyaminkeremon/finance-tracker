# Finance Tracker

A personal finance tracker for managing income and expenses. Add transactions, filter by type, category or month, and see a monthly breakdown chart. Data is saved in the browser using localStorage.

## Live Demo

[Try it here](https://bunyaminkeremon.github.io/finance-tracker/)

## What It Does

- Add income and expense transactions with description, amount, category and date
- Balance, total income and total expense summary
- Monthly bar chart comparing income vs expenses (Chart.js)
- Filter by transaction type, category, or month
- Delete transactions
- Data persists in the browser (localStorage)

## Tech Stack

- HTML5
- CSS3
- JavaScript (localStorage, DOM manipulation)
- Chart.js for data visualization

## Run Locally
```bash
git clone https://github.com/bunyaminkeremon/finance-tracker.git
```

Open `index.html` in your browser.
